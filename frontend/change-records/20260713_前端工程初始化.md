- 锁定 Next.js 16.2.10、React 19.2.7、Tailwind CSS 4.3.2 和 Motion 12.42.2 等实际安装版本；
- 新增 `dev`、`build`、`start`、`lint` 和 `typecheck` npm 脚本；

- 使用 npm `overrides` 将 Next.js 的 PostCSS 传递依赖更新到 8.5.19，消除安装时的中等级别安全提示；
- 保留现有分层目录和 `.gitkeep` 占位文件，未提前创建业务模块。
